Color system

Every colour job mapped to an official perceptually-uniform, colour-blind-safe scientific colour map — Crameri's batlow / vik / berlin and matplotlib's inferno — not a hand-rolled ramp. Because these maps run dark→light, each shows the background-matched variant so the informative end never sinks into the viewer, which now has a light and a dark mode.
